PowerTrade Login Help

Secure Sign-In & 2FA Guide for accessing your PowerTrade account.

Magic Link Sign-In

PowerTrade uses a passwordless magic link system for secure sign-in. Instead of remembering a password, you'll receive a one-time login link via email each time you sign in.

  1. Open app.power.trade or the PowerTrade mobile app
  2. Enter your registered email address
  3. Tap "Sign In"
  4. Check your email for the magic link (arrives within seconds)
  5. Click the link to be logged in automatically

Important: Magic links expire after 1 hour and can only be used once. If the link has expired, simply request a new one.

Two-Factor Authentication (2FA)

For additional account security, PowerTrade supports two-factor authentication. When enabled, you'll need to provide a second verification code after clicking the magic link.

  • Security PIN: A 4-digit PIN set during account creation that's required on each device
  • Authenticator App: Use Google Authenticator, Authy, or any TOTP-compatible app for time-based codes

Troubleshooting Login Issues

  • No email received? Check your spam/junk folder and whitelist support@power.trade
  • Link expired? Request a fresh magic link — they expire after 1 hour
  • Wrong device? Magic links must be opened on the same device/browser where you requested them
  • Web app issues? Try an incognito/private browsing window or clear your browser cache
  • Still stuck? Contact support@power.trade with your registered email

Login Security Best Practices

  • Always verify the URL is app.power.trade before entering credentials
  • Never share your magic link with anyone
  • Enable 2FA for maximum account protection

What Are Subaccounts?

Subaccounts provide the ability to segregate trading activity from your main account. Since PowerTrade accounts have portfolio margin enabled by default, creating subaccounts allows you to isolate your positions and margin balances for different strategies.

Benefits of Subaccounts

  • Strategy Isolation: Run different trading strategies independently with separate margin pools
  • Risk Management: Isolate high-risk positions from your main portfolio to prevent cross-contamination
  • Performance Tracking: Track individual strategy performance clearly without mixing P&L
  • Unlimited Subaccounts: Create as many subaccounts as needed at no additional cost

Creating a Subaccount

  1. Navigate to Settings → Subaccounts in the PowerTrade app
  2. Click "Create Subaccount"
  3. Give your subaccount a descriptive name (e.g., "BTC Options", "Altcoin Perpetuals")
  4. Your subaccount is ready to use immediately

Internal Transfers

Move funds between your main account and subaccounts instantly:

  • Free transfers — no blockchain fees for internal movements
  • Instant settlement — funds are available immediately
  • Margin impact — transfers affect the account health of both accounts, so plan accordingly

Note: Withdrawals can only be initiated from your main account. Transfer funds back to your main account before withdrawing.

Options Trading on PowerTrade

Learn to trade crypto options. Calls, puts, spreads & hedges for BTC, ETH and 80+ altcoin markets.

What Are Crypto Options?

Crypto options are financial derivatives that give traders the right, but not the obligation, to buy or sell an underlying cryptocurrency at a fixed price (the strike price) on or before a specific date.

Unlike spot trading or perpetual futures, options allow you to limit your downside while keeping unlimited upside potential. On PowerTrade, all options are European-style and settled in USDC.

Call Options

A call option gives the buyer the right (not the obligation) to buy the cryptocurrency at the strike price.

  • Traders buy calls when they expect the crypto price to rise
  • Profit potential is unlimited if the price moves above the strike price

Example: A trader buys a BTC call option with a $50,000 strike price. If BTC rises to $55,000 before expiry, the trader can buy at $50,000 and capture the $5,000 difference (minus the premium).

Put Options

A put option gives the buyer the right (not the obligation) to sell the cryptocurrency at the strike price.

  • Traders buy puts when they expect the crypto price to fall
  • Useful for hedging against downside risk

Example: A trader buys an ETH put option with a $3,000 strike price. If ETH drops to $2,500, the trader can sell at $3,000, profiting $500 (minus the premium).

European Style Options

PowerTrade offers European-style options, which can only be exercised at the expiration date. This structure:

  • Prevents early exercise risk
  • Simplifies strategy planning
  • Keeps the focus on price moves up to expiry

USDC Cash-Settled Options

All PowerTrade options are cash-settled in USDC:

  • No need to hold BTC, ETH, or altcoins directly
  • Settlement in stable USDC makes it easy to manage profits and losses
  • Reduces custody risk compared to physically settled options

Why Trade Options on PowerTrade?

  • ✅ 80+ altcoin options markets (the most in crypto)
  • ✅ Tight pricing & deep liquidity
  • ✅ Risk-defined strategies with capped downside
  • ✅ Flexible expirations — daily to 3 months forward

Crypto options are financial derivatives that give traders the right, but not the obligation, to buy or sell an underlying cryptocurrency at a fixed price (the strike price) on a specific date. European-Style Options

All options on PowerTrade are European-style, meaning they can only be exercised at the expiration date — not before. This structure:

All PowerTrade options are cash-settled in USDC. At expiration, the profit or loss is calculated and settled in USDC directly to your account:

Option Types

  • Call Options: Give you the right to buy at the strike price. Profit when the underlying price rises above the strike + premium paid.

Perpetuals Trading on PowerTrade

Trade perpetual contracts with no expiration date and deep liquidity.

What Are Perpetual Contracts?

Perpetual contracts (perps) are derivative contracts similar to futures but with no expiration date. They allow traders to speculate on the price of cryptocurrencies with leverage, without the need to roll over contracts at expiry.

Key Features

  • No expiration: Hold positions indefinitely without worrying about contract rollovers
  • Funding rate: Periodic payments between longs and shorts to keep the contract price anchored to the spot price
  • Leverage: Trade with leverage to amplify your exposure (and risk)
  • USDC-margined: All margin and settlement in USDC for simplicity

Funding Rate

The funding rate is exchanged between long and short position holders every 8 hours. When the rate is positive, longs pay shorts. When negative, shorts pay longs. This mechanism ensures the perpetual price stays close to the underlying spot price.

How to Trade Perpetuals

  1. Navigate to the Perpetuals section in the PowerTrade app
  2. Select the trading pair (e.g., BTC-PERP, ETH-PERP)
  3. Choose your position size and leverage
  4. Place a market or limit order
  5. Monitor your position and manage risk with stop-loss and take-profit orders

Futures Trading on PowerTrade

Trade crypto futures with fixed expiration dates, deep liquidity, and competitive fees.

What Are Crypto Futures?

Futures contracts are agreements to buy or sell a cryptocurrency at a predetermined price on a specific future date. Unlike perpetual contracts, futures have a fixed expiration date, making them ideal for precise timing strategies and hedging.

Key Features

  • Fixed expiration: Contracts settle on a specific date, providing certainty for hedging strategies
  • USDC-settled: All futures on PowerTrade settle in USDC
  • Leverage available: Trade with leverage to amplify exposure
  • Multiple expirations: Weekly, monthly, and quarterly expiration cycles

Futures vs. Perpetuals

  • Expiration: Futures expire on a set date; perpetuals never expire
  • Funding rate: Perpetuals have recurring funding payments; futures do not
  • Basis: Futures may trade at a premium or discount to spot (basis); perpetuals track spot via funding
  • Use case: Futures are better for calendar strategies and hedging specific time horizons

Contract Specifications

  • Settlement: USDC cash-settled at expiry
  • Trading fees: Maker 0.075% / Taker 0.075%
  • Margin: USDC collateral with portfolio margin

Margin & Liquidation

Understanding margin requirements and liquidation on PowerTrade.

Portfolio Margin

PowerTrade uses a portfolio margin system by default, which evaluates the risk of your entire portfolio holistically rather than on a per-position basis. This means hedged positions require less margin than isolated margin systems.

Margin Requirements

  • Initial margin: The minimum collateral required to open a new position
  • Maintenance margin: The minimum collateral required to keep a position open
  • Account health: Displayed as a percentage — when it drops to 0%, liquidation begins

Liquidation Process

When your account health reaches critical levels:

  1. Warning: You'll receive alerts when your account health is declining

Automate your crypto options trading with the PowerTrade API. Access markets, place orders, and manage positions programmatically.

Generate an API Key and Private Key

To start using the PowerTrade API, you'll need to generate an API key pair from the PowerTrade application:

  3. Click "Create New API Key"
  4. Save your api_key and private_key securely — the private key is only shown once

Important: Store your private_key securely. It cannot be retrieved after creation.

Generate an Access Token — JWT

The PowerTrade API uses JWT (JSON Web Tokens) signed with ES256 for authentication. Here's how to generate one:

import jwt
import time
import uuid

api_key = "your_api_key"
private_key = "your_private_key"

payload = {
    "sub": api_key,
    "iat": int(time.time()),
    "exp": int(time.time()) + 3600,
    "jti": str(uuid.uuid4())
}

jwt_token = jwt.encode(
    payload,
    private_key,
    algorithm="ES256"
)

print(f"Token: {jwt_token}")

  • REST API: https://api.power.trade/v1/
  • WebSocket: wss://api.power.trade/v1/ws

Include the JWT token in your request headers:

Authorization: Bearer <your_jwt_token>

API Authentication

Secure access to the PowerTrade Trading API using JWT tokens.

Authentication Overview

The PowerTrade API uses JWT (JSON Web Tokens) signed with the ES256 algorithm for authentication. Every authenticated API request must include a valid JWT token in the Authorization header.

API Key Management

  1. Log in to PowerTrade
  2. Navigate to Profile → API Keys
  3. Click "Create New API Key"
  4. Save your api_key and private_key securely

⚠️ Important: The private key is only displayed once at creation time. Store it securely — it cannot be retrieved later. If lost, delete the key and create a new one.

Generating a JWT Token

Use the ES256 algorithm to sign your JWT with the following payload structure:

import jwt, time, uuid

payload = {
    "sub": "your_api_key",
    "iat": int(time.time()),
    "exp": int(time.time()) + 3600,
    "jti": str(uuid.uuid4()),
    "client": "api"
}

token = jwt.encode(payload, private_key, algorithm="ES256")

Using the Token

Include the JWT token in every API request:

Authorization: Bearer <your_jwt_token>
Content-Type: application/json

Security Best Practices

  • Never expose your private key in client-side code or public repositories
  • Rotate API keys periodically
  • Set short token expiration times (1 hour recommended)
  • Use environment variables or secure vaults to store keys
  • Monitor API key usage and disable unused keys
